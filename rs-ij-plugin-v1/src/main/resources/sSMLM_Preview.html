RainbowSTORM sSMLM Preview

Spatial and Spectral Images

The spatial and spectral images after background subtraction of each localization are displayed on the preview screen for the first 10000 localizations.

Line Spectrum

The line spectrum of the single-molecule emission events from the spectral image after removing the background signals.

sSMLM Summary

A summary of the spectroscopic information described below for each localization are shown in the the preview screen.
  • Localization Uncertainty: the estimated uncertainty in the lateral position of the current fitted emission event (localization)
  • Photons(PSF): total photons of the current localization in the spatial domain
  • Spectral Uncertainty: the estimated uncertainty in the spectral centroid of the current localization
  • Photons (Spectrum):the total photons of the spectrum of the current localization
  • Percent of localizations: the number of localizations which were spectroscopically analyzed

The Update Event button

The RainbowSTORM settings(calibration information, background subtraction, and spectroscopic analysis parameters) can updated then the "Update Event" button can be pressed to load the localizations using the new processing parameters

The Next Event button

Pressing the "Next Event" button results in the calculation of the spectroscopic information for the next localization event using the current RainbowSTORM settings.

The Previous Event

Pressing the "Previous Event" button results in the calculation of the spectroscopic information for the next localization event using the current RainbowSTORM settings.
