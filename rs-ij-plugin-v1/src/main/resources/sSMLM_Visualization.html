RainbowSTORM sSMLM Visualization

RainbowSTORM Fields

Spatial information was estimated using ThunderSTORM.

  • id - Index of each fitted blinking event
  • frame - The frame of each blinking event
  • x - The subpixel x position of each blinking event. Used to reconstruction the super-resolution images. Values are confined within the bounds of the original image or selected region of interest. Estimated using ThunderSTORM.
  • y - The subpixel y position of each blinking event. Used to reconstruction the super-resolution images. Values are confined within the bounds of the original image or selected region of interest. Estimated using ThunderSTORM
  • z - The subpixel z position of each blinking event estimated using the astigmatism processing method using ThunderSTORM. Used to reconstruction the 3D super-resolution images. Accepted range: -2000 to 2000. spatial photons Total photons of the fitted blinking event in the spatial domain. Used to estimate the localization uncertainty. Estimated using ThunderSTORM
  • spatial sigma - The standard deviation of the fitted blinking event in the spatial domain. Used to estimate the localization uncertainty and point spread function width along the focal plane. Estimated using ThunderSTORM
  • spatial sigma1 - The standard deviation of the fitted blinking event in the spatial domain along the lateral axis for 3D sSMLM images. Used to estimate the localization uncertainty along the lateral axis and point spread function width along the focal plane. Estimated using ThunderSTORM
  • spatial sigma2 - The standard deviation of the fitted blinking event in the spatial domain along the axial axis for 3D sSMLM images. Used to estimate the point spread function width along the axial axis. Estimated using ThunderSTORM
  • localization uncertainty - The estimated uncertainty in the lateral position of each fitted blinking event. Combined with sample labeling, the localization uncertainty is used to estimate the resolution of the reconstructed image. Estimated using ThunderSTORM
  • spectral centroid - The spectral centroid or spectral mean of each blinking event. The centroid can be used to infer information related to the identity, local environment and conformational state of the source (dye or protein) of the blinking event. Values are used to color-code sSMLM images. Estimated using RainbowSTORM. Accepted centroids range from 380 to 900 with a minimum step size of 1
  • spectral photons - The total photons of the spectrum of each blinking event. Used to estimate the spectral uncertainty. Estimated using RainbowSTORM
  • spectral bk photons - The total background photons of the spectrum of each blinking event. Estimated using RainbowSTORM
  • spectral bk photons/pixel - The total background photons for each pixel of the image of the spectrum of each blinking event. Used to estimate the spectral uncertainty. Estimated using RainbowSTORM
  • spectral sigma - The standard deviation of the spectrum width. This value is estimated from fitting the spectrum to a Gaussian or using the average spectrum width for a given source. Used to estimate the spectral uncertainty. Estimated using RainbowSTORM.
  • spectral uncertainty- The estimated uncertainty in the spectral centroid of each blinking event. It can be used to exclude events whose spectral centroid cannot reliably be identified. Estimated using RainbowSTORM

Histograms combobox

The histograms of the localizations for each RainbowSTORM field can be generated by selecting the field and pressing the "Show Histogram" button.

Drift Correction File

RainbowSTORM can be used to load ThunderSTORM drift-correction JSON files. Pressing the "Correct Drift" button apply the drift correction to the spatial information for each localization

Visualization Settings

RainbowSTORM's visualization settings are used to set the reconstruction method (scatter plot, averaged gaussian image) and magnification for the image reconstruction. Reconstruction Methods:
  • Scatter Plot: Super-resolution reconstruction generated from equally weighted points representing the x and y position of each localization
  • Averaged Gaussian: Super-resolution reconstruction generated from 2D Gaussians representing the x and y positions of each localization where the radius of each Gaussian corresponds to the localization uncertainty.

Filters

The localizations can be filtered by specifying the start and end value for the accepted range of each RainbowSTORM field selected from the Filter combo box.

Restrict to ROI button

ImageJ's rectangular selection tool is used to select a region of interest(ROI) in the sSMLM reconstruction then pressing the "Restrict to ROI" button to select localizations within the ROI

Display 3D sSMLM button

The "Display 3D sSMLM" button is only available for 3D sSMLM images acquired using the astigmatism method. astigmatism method. Pressing this button renders a stack of images color-code by their spectral centroid with each image in the stack based on the axial information for each localization.

Classify by Centroid button

The "Classify by Centroid" button launches the RainbowSTORM Classification screen where localizations can be grouped into color-coded image windows based on their spectral centroids.

Calculate FRC button

Pressing the "Calculate FRC" button performs Fourier Ring Correlation (FRC) analysis for the sSMLM super-resolution reconstruction. FRC analysis takes spatial sampling and localization uncertainty into consideration when determining the resolution of super-resolution reconstructions. The resulting FRC plot and Fourier Image REsolution(FIRE) number are generated.

Reset sSMLM Data button

Pressing the "Reset sSMLM Data" button removes any RainbowSTORM filters or drift-correction which have been applied to the data.

sSMLM Summary

  • Post-processing: provides a summary of the operations which have been applied(e.g drift-correction or any RainbowSTORM filters)
  • Average spectral photons: the average photons in the spectral domain for the localizations currently displayed
  • Average spectral uncertainty: the average estimated uncertainty in the spectral centroid of localization currently displayed
  • Number of localizations: the number of localizations currently displayed
  • Percent of spectroscopic localizations: the percent of the localizations currently displayed relative to all localizations with spectroscopic information
  • Percent of SMLM localizations: the percent of the localizations currently displayed relative to all spatially localized emission events

Save Spectroscopic Data

The spectroscopic data currently displayed can be saved as two CSV file for future analysis. The first file saves all RainbowSTORM fields and the second file includes all RainbowSTORM fields as well as the values for the full spectrum range. RainbowSTORM appends the "_spec" to the second file which can be loaded into RainbowSTORM using the Import screen.

Lambda-Colored Images and colorbar

Lambda-colored super-resolution reconstructions are color-coded using the spectral centroids of each localization to set a color-range
