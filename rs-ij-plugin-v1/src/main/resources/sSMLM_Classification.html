RainbowSTORM Classification

Predefined color channels

Up to six (6) color ranges can be assigned to predefined color channels based on the centroid distribution. The "Show Channel" checkbox can be selected to show each single-color channel.

Show Channel Summaries checkbox

The channel summary screen displays a scatter plot of the centroids versus photons and a histogram of the centroid distribution with the channels highlighted. The channel summary is optional and can be deselected using the "Show Channel Summary" checkbox.

Classify by Centroid button

Pressing the "Classify by Centroid" button generates a multicolor image of the selected data as well as any single-color channel reconstructions.
