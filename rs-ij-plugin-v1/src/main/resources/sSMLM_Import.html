RainbowSTORM sSMLM Import

Load saved sSMLM Data

Previously saved data files can be loaded into RainbowSTORM by selecting the file ending in "_spec.csv".

Camera Setup Parameters

The camera setup parameters can be manually input or the previously saved parameters can be loaded.

The Visualize Data button

Pressing the "Visualize Data" button renders the Pseudo-colored image and loads the data into the Visualization Screen.
