RainbowSTORM sSMLM Calibration

Uncalibrated Spectrum Plot:

The uncalibrated spectrum plot shows the average line plot from the calibration image. Double-click on the red uncalibrated line plot starting with the first major peak which represents the spatial image pixel position.

Calibrated Spectrum Plot:

The calibrated spectrum plot shows the resulting line plot after calibration.

Pixel vs Wavelength Plot:

The pixel versus wavelength plot shows the relationship between the pixels and wavelengths as well as the fitting result after calibration

Current Peak Text field:

The current peak text field shows the value from the uncalibrated spectrum line plot. This field can also be manually updated.

Peak Position text fields:

The spatial and spectral peak position text fields can be updated with the peak pixel values from the current peak text field by pressing the set button or manually. Only one text field is enabled for editing at a time. The enabled text field is highlighted in green.

Set button:

The Set button can be pressed to update the enabled pixel position text field. The value from the current peak position text field value will be transfered to the enabled pixel position text field. The updated text field will be disabled and the next text field will be enabled.

Next button:

The Next button allows the user to enable the next peak position text field.

Previous button:

The Previous button allows the user to enable the previous peak position text field.

Slit/Emitter Position:

Starting position for he spatial information.

Wavelength Text Fields:

Wavelength information corresponding to each uncalibrated spectral peak.

Calibration Source Combobox:

RainbowSTORM allows for calibration using a calibration light source (calibration lamp or multiple laser lines) or a single multi-color fluorescent bead.

Dispersive Element Combobox:

Grating-based or prism-based systems can be calibrated using RainbowSTORM. Calibration for grating-based systems uses linear fitting. Calibration of prism-based systems can be done using either second-order or third-order fitting.

Calibrate Button:

Pressing the calibrate button performs calibration for the selected calibration source type and dispersive element.

Save Button:

The save button is pressed to save the fitting method along with the peak pixel positions and their corresponding wavelength information as a CSV file.
